## Вариант
Вариант 2 — Шифр Виженера.

## Описание
Шифр Виженера — полиалфавитный шифр, использующий ключевое слово.
Каждая буква исходного текста сдвигается на позицию соответствующей буквы ключа.
Ключ автоматически повторяется до длины текста.

Пробелы, цифры и знаки препинания не шифруются и остаются на своих местах.

## Реализованные методы
- EncryptVigenere(string text, string key)
- DecryptVigenere(string cipher, string key)

## Проверки
- пустой текст;
- пустой ключ;
- ключ с недопустимыми символами;
- работа с пробелами и знаками препинания;
- ключи разной длины;
- известный пример ATTACKATDAWN + LEMON = LXFOPVEFRNHR.
